图片超分辨率工具 v1.1.1

推荐使用方式：
1. 从 GitHub Latest Release 下载“图片超分辨率工具-v1.1.1-安装器.exe”。
2. 双击安装器。
3. 选择安装位置。
4. 安装完成后，用桌面快捷方式打开软件。
5. 第一次使用先点软件左下角的“安装/检查环境”。
6. 等环境安装完成后，选择图片和输出文件夹，然后开始超分。

模型说明：
- ATD 官方超分辨率模型：默认模型，恢复之前 Transformer 版的效果。
- HAT 高质量超分辨率模型：细节和锐度较强，但速度更慢、显存占用更高。
- Real-ESRGAN 通用照片模型：速度和效果比较均衡。
- Real-ESRGAN 动漫插画模型：适合动漫、插画、二次元图片。

说明：
- 安装器已经包含上面四个模型，不需要用户单独下载模型。
- 安装器不包含 Python 运行环境和示例照片。
- CUDA 版 PyTorch 和其它依赖会通过软件内的可视化安装窗口自行下载。
- 如果电脑没有 Python，启动软件时会打开 Python 下载页。安装 Python 后重新打开软件即可。
- RTX 4060 8GB 推荐使用默认 ATD；如果处理速度太慢，可以切换到 Real-ESRGAN 通用照片模型。

推荐配置：
- Windows 10 / Windows 11
- NVIDIA RTX 4060 8GB 或更高
