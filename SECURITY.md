## 下载安全建议

- 建议只从项目发布页或作者提供的可信位置下载单文件安装器。
- 单文件安装器包含 ATD 官方超分辨率模型、HAT 高质量超分辨率模型、Real-ESRGAN 通用照片模型和 Real-ESRGAN 动漫插画模型，但不包含 `.venv` 运行环境。
- 第一次使用时，通过软件内的 `安装/检查环境` 可视化窗口下载 CUDA 版 PyTorch 和其它依赖。
- 如果电脑没有 Python，先从 Python 官方网站安装 Python 3.12，再打开软件。
- 不建议从第三方网盘或未知来源下载改包版本。

## 报告安全问题

如果发现安全问题，请不要把可利用细节直接公开到普通 Issue。可以先创建一个不含敏感细节的 Issue，说明“发现安全问题，需要私下沟通”。
