# 常见问题

## 为什么现在的包很小？

这是轻量版。它不包含 Python 运行环境、CUDA 依赖、模型权重和示例照片。用户第一次使用时，通过软件内的 `安装/检查环境` 自行下载运行环境。

## 环境从哪里下载？

Python 会通过 winget 或 python.org 获取。CUDA 版 PyTorch 会从 PyTorch 官方源获取。其它依赖从 Python 包源获取。模型权重会在第一次超分时下载到 `models` 文件夹。

## 可以不上传运行环境吗？

可以。当前 GitHub 版本就是轻量包，仓库和 Release 都不上传 `.venv` 环境和模型权重。

## 为什么第一次安装慢？

CUDA 版 PyTorch 体积较大，第一次安装需要下载较多文件。安装完成后，后续打开软件不需要重复下载环境。

## 推荐什么显卡？

推荐 NVIDIA RTX 4060 8GB 或更高。软件会自动识别显卡型号、显存和空闲显存，并按性能调整处理占用。

## 会不会把我的照片上传到 GitHub？

不会。仓库和轻量包不包含用户照片。程序也不会保存上次输入图片路径和输出文件夹路径。

## 为什么删除了预览栏？

为了避免打开软件时显示上次处理过的图片，也让界面更干净。处理完成后可以直接到输出文件夹查看结果。
