# 常见问题

## 这个安装器为什么比较大？

因为 `v1.1.2` 安装器已经包含四个超分模型。这样用户不需要再单独下载模型。

## 还需要安装环境吗？

需要。安装器包含模型和可直接打开软件界面的启动器，但不预置 `.venv` 运行环境。第一次使用时，通过软件内 `安装/检查环境` 安装本地 Python 运行时、CUDA 版 PyTorch 和其它依赖。

## 一键安装怎么用？

下载 Release 里的 `ImageSuperResolutionTool-v1.1.2-Setup.exe`，双击运行，然后选择安装位置。安装完成后桌面会出现快捷方式。

## 会不会把我的照片上传到 GitHub？

不会。仓库和发布包不包含用户照片。程序也不会保存上次输入图片路径和输出文件夹路径。

## 推荐什么显卡？

推荐 NVIDIA RTX 4060 8GB 或更高。软件会自动识别显卡型号、显存和空闲显存，并按性能调整处理占用。

## 模型还需要单独下载吗？

不需要。安装器已经包含 ATD、HAT、Real-ESRGAN 通用照片和 Real-ESRGAN 动漫插画四个模型。
