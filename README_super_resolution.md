# 图片超分辨率工具

这是一个无需写代码的 Windows 图形界面工具。它会自动识别 NVIDIA 显卡型号、显存容量和当前空闲显存，然后按显卡能力自动匹配显存占用、推理精度和 tile 大小。

默认模型已经改为 Transformer 路线：`ATD x4 Transformer`。这个模型适合做高质量单张图片超分，比普通拉伸和传统 CNN 模型更重，也更依赖显卡显存。

说明：它不是游戏里的原生 DLSS。原生 DLSS 需要游戏引擎提供多帧、运动向量和深度信息；这个工具处理的是单张图片或图片文件夹。

## 第一次使用

1. 双击 `安装运行环境.bat`。
2. 等它自动安装 Python、创建 `.venv` 专用环境、安装 CUDA 版 PyTorch 和 Transformer 模型加载库。
3. 看到 `CUDA available: True` 后，关闭安装窗口。
4. 双击 `打开超分工具.bat`。

安装过程可能比较久，因为 CUDA 版 PyTorch 文件较大。

## 日常使用

1. 双击 `打开超分工具.bat`。
2. 点击“选择图片”或“选择文件夹”。
3. 选择输出位置。
4. 保持默认设置，点击“开始超分”。

第一次超分时会自动下载模型权重到 `models` 文件夹，后续会直接使用本地模型。

底部进度条会显示当前处理百分比、已用时间和预计剩余时间。勾选“简洁日志”只会减少日志文字，不会影响进度条显示。

## 自动显卡匹配

程序会自动读取：

- 显卡型号
- CUDA 能力
- 总显存
- 当前空闲显存
- SM 数量

然后自动设置：

- `precision`：程序按模型自动选择；Real-ESRGAN 通常使用 fp16，ATD Transformer 会在需要时使用 fp32。
- `tile`：Transformer 模型默认按显存保守匹配。
- `tile_pad`：自动设置边缘缓冲，减少拼接痕迹。
- 显存占用上限：默认留出一部分显存给桌面和系统。

如果显存仍然不足，程序会自动降低 tile 后重试。

## 显卡占用选项

- 自动：推荐，程序按显卡自动选择。
- 保守：占用更低，适合同时开着其他软件。
- 均衡：稳定和速度折中。
- 性能：占用更高，速度更快，但更容易显存不足。

RTX 4060 8GB 建议使用“自动”。

## 自动锐化和去噪

默认启用自动锐化和自动去噪。程序会分析原图的噪声、边缘强度和纹理密度，每张图单独决定后处理强度。

如果想手动控制，可以取消“自动锐化”或“自动去噪”，再拖动对应滑条。

## 模型选择

- ATD Transformer 自动推荐：默认模型，质量优先。
- Real-ESRGAN 照片备用：照片、截图、游戏画面备用。
- Real-ESRGAN 动漫备用：动漫、插画、线稿备用。

## 常见问题

如果安装后仍显示 CUDA 不可用，请更新 NVIDIA 显卡驱动，然后重新运行 `安装运行环境.bat`。

如果第一次超分很慢，通常是在下载模型；第二次会快很多。

如果输出文件很大，建议选择 2 倍或 3 倍放大，而不是 4 倍。
