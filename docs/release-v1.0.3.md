## 图片超分辨率工具 v1.0.3

这是轻量包、隐私和体验修正版。

### 修复内容

- 软件不再保存上次选择的输入图片路径。
- 软件不再保存上次选择的输出文件夹路径。
- 打开软件时不再自动显示上次处理过的图片。
- 删除软件内处理前后预览栏。
- 轻量包不包含 `.venv` 环境和模型权重，用户通过软件内 `安装/检查环境` 自行下载环境。
- 软件安装环境改为 `.bat` 和 Python 脚本。
- 环境安装完成后，当前界面也会优先使用新建的 `.venv` 环境运行超分任务。

### 下载方式

轻量包下载后双击 `打开超分工具.bat`。首次使用先点软件左下角的 `安装/检查环境`，由用户电脑自行下载 Python、CUDA 版 PyTorch、其它依赖和模型权重。
