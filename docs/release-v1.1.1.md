## 图片超分辨率工具 v1.1.1 单文件安装器版

这是当前推荐公开版本。普通用户只需要下载一个安装器文件：

`图片超分辨率工具-v1.1.1-安装器.exe`

### 变化

- 发布方式改为单文件安装器，不再提供 zip 包和单独下载脚本。
- 安装器启动后可以选择安装位置。
- 安装完成后自动创建桌面快捷方式。
- 安装器内包含四个模型：
  - ATD 官方超分辨率模型
  - HAT 高质量超分辨率模型
  - Real-ESRGAN 通用照片模型
  - Real-ESRGAN 动漫插画模型
- 清理旧版下载器、独立环境安装脚本和旧启动入口。

### 使用方式

下载并双击 `图片超分辨率工具-v1.1.1-安装器.exe`，选择安装位置。安装完成后，使用桌面快捷方式打开软件。

第一次打开软件后，点击左下角 `安装/检查环境` 安装 CUDA 版 PyTorch 和其它依赖。
