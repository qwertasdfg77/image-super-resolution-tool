## 图片超分辨率工具 v1.0.4 含模型版

这是默认模型内置版本。

### 修复内容

- 包内加入默认 Transformer 模型 `003_ATD_SRx4_finetune.pth`。
- 默认超分不再需要首次联网下载模型。
- 保留软件内环境安装按钮，用户仍可自行安装 Python、CUDA 版 PyTorch 和其它依赖。
- 说明文件改为含模型版口径，避免误写“轻量包不含模型”。

### 使用方式

下载后双击 `打开超分工具.bat`。首次使用先点软件左下角的 `安装/检查环境`，环境安装完成后即可开始超分。

### 注意

如果手动选择照片/动漫备用模型，对应备用模型仍会按需下载。默认 Transformer 模型已经包含在包内。
