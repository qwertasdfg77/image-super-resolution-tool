# 图片超分辨率工具轻量版

这个包不包含 `.venv` Python 运行环境，也不包含模型权重，所以体积更小。

## 使用方法

1. 双击 `打开超分工具.bat`。
2. 如果软件能打开，先点左下角的 `安装/检查环境`。
3. 等环境安装完成后，再选择图片并开始超分。

如果电脑没有 Python，双击 `打开超分工具.bat` 时会自动打开安装窗口。也可以直接双击 `安装运行环境.bat`。

## 会下载什么

- Python 3.12：从 winget 或 python.org 下载。
- CUDA 版 PyTorch：从 PyTorch 官方源下载。
- 其它依赖：从 Python 包源下载。
- 模型权重：第一次超分时自动下载到 `models` 文件夹。

安装过程使用 `.bat` 和 Python 脚本。

## 注意

第一次安装会比较慢，因为显卡版 PyTorch 很大。安装完成后，后续打开软件不需要重复下载环境。
