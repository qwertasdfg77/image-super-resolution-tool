# 更新记录

## v1.1.3 - 2026-06-08

运行环境安装修复版。

- 运行环境改为安装到当前用户的本地数据目录，避免桌面、Program Files 或受保护安装目录写入失败。
- Python 安装器失败时会重新下载并重试。
- Python 安装过程增加日志文件路径，方便定位另一台电脑上的安装失败原因。
- 如果 Python 官方安装器把同版本 Python 安装到默认目录，软件会自动识别并继续创建运行环境。
- 启动脚本兼容新的本地运行环境目录，同时保留旧 `.venv` 目录兼容。

## v1.1.2 - 2026-06-08

单文件安装器版。

- 发布资产改为 `ImageSuperResolutionTool-v1.1.2-Setup.exe`。
- 安装器支持选择安装位置。
- 安装完成后自动创建桌面快捷方式。
- 安装器内包含 ATD、HAT、Real-ESRGAN 通用照片、Real-ESRGAN 动漫插画四个模型。
- 软件窗口标题继续在尾部显示当前版本号。
- 模型下拉框顺序为：ATD、HAT、Real-ESRGAN 通用照片、Real-ESRGAN 动漫插画。
- 修复无 Python 电脑点击桌面快捷方式时打开 Python 官网的问题。
- 软件界面可先打开，再通过 `安装/检查环境` 安装本地 Python 运行时和 CUDA 依赖。
- 删除旧的 zip 下载器、单独环境安装脚本和旧启动入口。
