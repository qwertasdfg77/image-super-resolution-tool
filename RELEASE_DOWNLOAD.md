# 下载说明

普通用户只需要下载 Latest Release 里的 `ImageSuperResolutionTool-Lightweight.zip`。

Latest Release:
https://github.com/qwertasdfg77/image-super-resolution-tool/releases/latest

## 推荐方式

1. 下载 `ImageSuperResolutionTool-Lightweight.zip`。
2. 解压到桌面或其它文件夹。
3. 双击 `打开超分工具.bat`。
4. 第一次使用先点 `安装/检查环境`。

## 可选下载器

Release 里也提供 `download-lightweight.bat`。它会把最新轻量包下载到桌面并解压。

## 当前版本

`v1.0.3` 是轻量包、隐私和体验修正版：

- 不包含运行环境和模型权重。
- 不包含示例照片。
- 不保存上次输入图片路径和输出文件夹路径。
- 删除软件内处理前后预览栏。
- 环境安装完成后，当前界面会优先使用新建的 `.venv` 环境运行超分任务。
