# 下载说明

普通用户推荐下载 Latest Release 里的 `download-and-install.bat`，双击即可下载、解压并安装运行环境。

Latest Release:
https://github.com/qwertasdfg77/image-super-resolution-tool/releases/latest

## 推荐方式

1. 下载 `download-and-install.bat`。
2. 双击运行。
3. 等待下载和解压完成。
4. 按安装窗口提示完成环境安装。
5. 打开软件后选择图片开始超分。

## 手动方式

1. 下载 `ImageSuperResolutionTool-ModelIncluded.zip`。
2. 解压到桌面或其它文件夹。
3. 双击 `打开超分工具.bat`。
4. 第一次使用先点 `安装/检查环境`。

## 当前版本

`v1.0.4` 是含模型版：

- 包内包含默认 Transformer 模型。
- 默认超分不需要首次联网下载模型。
- 不包含 `.venv` Python 运行环境。
- 不包含示例照片。
- 不保存上次输入图片路径和输出文件夹路径。
- 删除软件内处理前后预览栏。
